# BreakBite 🍱

> Skip the queue. Pre-book your canteen slot.

A full-stack web app for CDAC students to pre-book 
canteen slots during lunch and tea breaks.
Canteen staff see real-time demand to reduce 
food waste and long queues.

## Tech Stack
- Frontend: React (Vite) + Tailwind CSS + Framer Motion
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Auth: JWT + bcrypt
- Deploy: Vercel (frontend) + Render (backend)

## Features (v1)
- Student signup/login
- View today's menu
- Book a time slot
- Unique token per booking
- Canteen admin dashboard
- Real-time prep list
